Se7enSquare — Demo Videos
=========================

Place your recorded demo videos here before submission.

Suggested recordings:
1. backend-crud-demo.mp4       — Swagger UI showing GET/POST/PATCH/DELETE for all modules
2. admin-dashboard-demo.mp4    — Admin login → User CRUD → Community management → Events
3. mod-panel-demo.mp4          — Moderator login → Report queue → Resolve report via backend
4. discovery-events-demo.mp4   — Gamer login → Browse communities → View events
5. report-submit-demo.mp4      — Gamer login → Submit a report → Confirm backend receives it

Recording Tips:
- Start the NestJS backend first: cd back-end && npm run start:dev
- Open http://localhost:3000/api/docs to show Swagger documentation
- Keep each video under 5 minutes
- Show the browser console (F12) to demonstrate API calls firing
